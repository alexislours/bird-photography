# Darktable

[Darktable](https://www.darktable.org/) is an open source RAW developer and Lightroom Classic alternative.

{% hint style="success" %}
Pros:

* Free
* Runs on any platform
* Open source
{% endhint %}

{% hint style="warning" %}
Cons:

* Limited image editing
* Not as feature complete as Lightroom Classic
{% endhint %}
